Лабораторная работа: Написание функций в СУБД
Цель работы
Освоить создание и использование пользовательских функций в реляционной СУБД. Научиться писать функции на языке SQL и процедурном расширении, использовать параметры, возвращать скалярные значения и табличные результаты.
Теоретические сведения
Функция в СУБД — это именованный объект базы данных, который принимает параметры, выполняет определённые действия и возвращает результат. Функции могут быть:
•	скалярными — возвращают одно значение (число, строка, дата и т.д.);
•	табличными — возвращают набор строк (как результат запроса);
•	void — не возвращают значения (аналог процедуры).
Преимущества использования функций:
•	повторное использование кода;
•	централизованная бизнес-логика;
•	повышение производительности за счёт выполнения на стороне сервера;
•	безопасность (можно ограничить прямой доступ к таблицам, разрешив только вызов функций).

Задания
Задание 1. Скалярная функция: 
Создайте функцию, которая  принимает название и возвращает количество относящиеся к этому названию в таблице записей. (например принимает название группы и возвращает количество студентов в этой группе).
Задание 2. Скалярная функция: средний значение
Создайте функцию, которая возвращает среднее значение (например оценку студента по всем предметам). Результат округлите до двух знаков после запятой.
Задание 3. Табличная функция: список студентов с задолженностями
Создайте функцию, которая возвращает таблицу со списком, (например  имеющих хотя бы одну оценку 2 (неудовлетворительно)). Выведите как минимум 3 столбца из таблицы.
Задание 4. Функция с проверкой: 
Создайте функцию,  которая добавляет запись в таблицу  только в том случае, если запись не существует. Функция должна возвращать булево значение: TRUE – если добавление выполнено, FALSE – если произошла ошибка.
Задание 5. Функция с проверкой: 
Создайте функцию,  которая удаляет запись в таблице. Функция должна возвращать булево значение: TRUE – если добавление выполнено, FALSE – если произошла ошибка.
