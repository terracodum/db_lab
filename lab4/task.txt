Лабораторная работа: Создание форм для работы с базами данных в C# (Windows Forms)
Цель работы: Изучить технологию создания форм для работы с базами данных в среде Visual Studio с использованием языка C# и Windows Forms. Научиться использовать мастер настройки источников данных, создавать главные и подчинённые (связанные) формы, а также редактировать элементы управления в режиме конструктора.
Подготовка к работе
1.	Откройте Visual Studio и создайте новый проект Windows Forms App (.NET Framework или .NET Core/5+).
2.	Подключитесь к базе данных. Для этого можно использовать:
o	Файл Microsoft Access (.accdb или .mdb) через провайдер OleDb;
o	Базу данных SQL Server (LocalDB) или другую СУБД.
3.	Добавьте источник данных в проект:
o	В меню выберите Данные → Добавить новый источник данных.
o	Выберите База данных → Далее.
o	Укажите подключение к вашей БД (например, через файл Access или строку подключения к SQL Server).
o	Выберите нужные таблицы (например, «Типы» и «Товары») и нажмите Готово.
o	В результате будет создан DataSet с таблицами и, при необходимости, связи между ними (если они заданы в БД).
1. Создание формы с помощью «мастера» (автоматическая генерация)
В Visual Studio аналогом мастера форм Access является возможность перетаскивания таблиц из окна Источники данных (Data Sources) непосредственно на форму. При этом автоматически создаются элементы управления с привязкой к данным.
Пошаговая инструкция:
1.	Убедитесь, что окно Источники данных открыто (меню Данные → Показать источники данных).
2.	Выберите таблицу (например, Типы), которую хотите отобразить в форме.
3.	Нажмите на стрелку рядом с именем таблицы и выберите способ отображения:
o	DataGridView — для табличного представления;
o	Подробности — для создания набора текстовых полей (по одному на каждое поле записи).
4.	Перетащите таблицу из окна Источники данных на пустую форму.
o	Visual Studio автоматически добавит на форму:
	BindingSource (связь с таблицей);
	TableAdapter (для заполнения данных);
	BindingNavigator (полоска с кнопками навигации, добавления, сохранения);
	Набор текстовых полей (если выбран режим «Подробности») или DataGridView (если выбран табличный режим).
5.	Запустите проект (F5) — данные загрузятся автоматически, и вы сможете просматривать записи.
Итог работы: Вы должны создать приложение Windows Forms, которое позволяет просматривать и редактировать данные из базы данных, а также реализовать связанный просмотр данных в главной подчинённой форме.
